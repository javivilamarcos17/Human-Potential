# Departamentos — tu panel de mandos para hablar con Claude

> Cada documento es el "puente" entre una persona de negocio y el código. Te dice qué
> controla ese departamento, qué **diales** puedes pedir que cambie (con su valor actual),
> dónde vive en el código y **cómo pedírmelo en una frase**. No necesitas saber programar:
> abres el documento de tu área, me dices "sube esto / cambia aquello" y yo lo aplico.

| Departamento | Qué decide | Documento |
|--------------|-----------|-----------|
| 🏋️ Entrenamiento | Rutinas, packs, cómo se adapta el ejercicio, progresión, XP | [entrenamiento.md](entrenamiento.md) |
| 🩺 Fisioterapia | Triaje, escalas de dolor, banderas rojas, Auditoría, semáforos | [fisioterapia.md](fisioterapia.md) |
| 🥗 Nutrición | Hábitos de vida, consejos (marco para integrar) | [nutricion.md](nutricion.md) |
| 📣 Marketing y Comercial | Viralidad, referidos, descuentos, centros, tono | [marketing-comercial.md](marketing-comercial.md) |

**Cómo trabajar:** dime, por ejemplo, *"en entrenamiento, añade un ejercicio al Pack Lumbar"*
o *"en fisio, baja el umbral del semáforo rojo a 7"*. Yo abro el dial correcto y lo cambio.
El motor del MVP (`mockups/prototipo/app-funcional.html`) está hecho a propósito con todas
estas reglas en tablas editables, para que cambiarlas sea rápido y sin riesgo.
